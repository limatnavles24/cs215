\left|\mu - v\right| = \left|\frac{\sum_{i=1}^{N}x_i - v}{N}\right| \leq \frac{\sum_{i=1}^{N}\left|x_i - v\right|}{N}  by triangular inequality
\frac{\sum_{i=1}^{N}\left|x_i - v\right|}{N} \leq \frac{\sum_{i=1}^{N}\left|x_i - \mu \right|}{N} 
x = median is the value for which \frac{\sum_{i=1}^{N}\left|x_i - x\right|}{N} is minimum
now, {\Bigg(\frac{\sum_{i=1}^{N}\left|x_i -\mu\right|}{N}\Bigg)}^2 \leq \frac{\sum_{i=1}^{N}{\left|x_i -\mu\right|}^2}{N} = \sigma^2 by (rms- am inequality)
\therefore \left|\mu - v\right| \leq \sigma
